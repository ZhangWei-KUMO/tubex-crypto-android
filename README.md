# TubeX 币圈风控模型Android客户端

这是一个原生Android应用，针对加密货币市场，本应用集成了各类统计学模型和机器算法模型；
本项目主要用于统计学学习研究，仅供学习只用，不做任何投资建议。

本项目尚处于开发阶段。
